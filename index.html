Terminal Copy

  • Ghostty: Cmd+Shift+C copies without trailing soft linebreaks
  • iTerm2: Edit → Copy Without Newlines at End
  • Enable "Unwrap soft line breaks" above if lines were wrapped at ~80 chars

Pasting into Slack

  • Rich Text paste: Slack renders bold, italic, code inline
  • mrkdwn paste: Slack renders formatting after you send
  • Shift+Enter — newline without sending in Slack
  • Rich Text paste works best in Slack desktop app

Conversion Notes

  • Tables → code blocks (Slack has no table support)
  • Headers → *bold text* (Slack has no heading syntax)
  • Images → links; code fence language tags are stripped
  • Task lists: - [ ] → ☐, - [x] → ☑

Claude Code Tips

  • Copying from claude.ai web gives cleaner markdown than terminal
  • Cmd+A in the input area to select all, then paste over
  • Use "Strip leading whitespace" if every line has an extra space
